配置项

Title Attributes

属性 说明 类型 是否必输？ 默认值 备注
title 标题内容 string 否 ""
fixed 是否固定标题头 boolean 否 true 如果设置为false，页面内容过多的时候，标题头就会一起上下滚动
backicon 返回按钮图片 string 是

隐藏原生标题头

最后，别忘了隐藏原生标题头。直接在 pages.json文件 的全局配置属性 globalStyle 或页面的当前页配置属性 style 中添加如下配置即可： 

{
    "navigationStyle": "custom"
}
