使用

页面中准备好以后，直接使用open方法即可打开： 

    myinputVercodeRef.value.open(config).then((value) => {
    // value就是用户输入的验证码
}).catch((err)=>{
    // 如果出错或者用户取消了选择
});

其中config的格式如下： 

{
    "phone": string
}
