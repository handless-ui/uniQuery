使用 



export default {
    data() {
        return {
            list: [
                "关注", "推荐", "电影", "电视剧", "小视频", "游戏", "校园", "影视"
            ]
        };
    },
    methods: {
        change(item) {
            console.log(item);
        }
    }
};

比如点击"推荐"的时候，打印内容就是： 

{
    index: 1, 
    value: "推荐"
}

配置项

Tabs Attributes

属性 说明 类型 是否必输？ 默认值 备注
index 初始化选中序号 number 否 0
list 列表内容 Array<string> 是 例如：["关注", "推荐", "电影"]

Tabs Events

名称 说明 参数 备注
change 初始化或用户点击切换选中条目的时候触发 { index: number, value: string }
