使用

页面中准备好以后，直接使用open方法即可打开： 

myselectorRef.value.open(getList).then((value)=>{
    // value就是用户选择的值
}).catch((err)=>{
    // 如果出错或者用户取消了选择
});

其中getList是一个函数，用以根据当前选择的内容获取下一级可选列表，比如： 

let getList = (selected)=>{
    // 根据selected值返回，下面为了演示写死
    return [
        { label: "江苏省", value: "js" },
        { label: "浙江省", value: "zj" },
        { label: "山东省", value: "sd" }
    ];
}

上面的 selected 是一个数组，比如 ["js","nj"] 表示前面选择的是 江苏省南京市 ，现在需要返回区列表。

此外，如果用户希望下次打开的时候，默认选中上次的结果，可以： 

let oldValue = undefined;
myselectorRef.value.open(getList, oldValue).then((value)=>{
    oldValue = value;
});
