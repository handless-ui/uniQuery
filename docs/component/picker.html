使用

页面中准备好以后，直接使用open方法即可打开： 

mypickerRef.value.open(lists).then((value)=>{
    // value就是用户选择的值
}).catch((err)=>{
    // 如果出错或者用户取消了选择
});

选择分为单列和多列，下面分别举例子：

单列 

let oldValue = undefined;
let lists = [
    { label: "数学", value: "mathmatical" }, 
    { label: "英语", value: "english" }, 
    { label: "物理", value: "physics" }, 
    { label: "地理", value: "geography" }, 
    { label: "语文", value: "chinese" }, 
    { label: "生物", value: "biology" }, 
    { label: "历史", value: "history" }
];

mypickerRef.value.open(lists, oldValue).then((value) => {
    oldValue = value;
});

多列
