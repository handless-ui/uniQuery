使用

页面中准备好以后，直接使用open方法即可打开： 

this.$refs.mycalendar.open().then((value) => {
    // value就是用户选择的值
}).catch((err)=>{
    // 如果出错或者用户取消了选择
});

其中value的格式如下： 

{
    "year": number
    "month": number
    "day": number
}

此外，open方法可以传递一个参数value，表示当前选中的日期： 

this.$refs.mycalendar.open(value);
特别说明：为什么当前选中的日期由open传递而不是直接通过组件属性配置？这是为了提高组件的复用度，保证多个输入可以复用一个日历组件（别的地方类似设计理由相同，不再赘述）。
