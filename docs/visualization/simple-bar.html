使用

let axis = ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'];
let data = [120, 200, 150, 80, 70, 110, 130];

配置项

SimpleBar Attributes

属性 说明 类型 是否必输？ 默认值 备注
label 非数值刻度列表 Array<string> 是
data 数值清单 Array<number> 是
labelType 非数值轴 string 否 "xAxis" 可以是"xAxis"或"yAxis"
color 柱状图颜色 string 否 "rgb(84,112,198)"
