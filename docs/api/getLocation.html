使用 

uni.$uniQuery.getLocation().then(res=> {
    // 获取成功
}).catch(err=>{
    // 获取失败
});

其中，res的数据格式如下： 

{
    longitude: number // 经度
    latitude: number // 纬度
}

权限相关

微信小程序

需要提前在 manifest.json 中进行权限申请的注册： 

"mp-weixin": {
    "permission": {
        "scope.userLocation": {
            "desc": "您的位置信息将用于小程序页面"
        }
    },
    "requiredPrivateInfos": [
        "getLocation"
    ]
}

微信公众号

需要在 jsApiList 补充如下方法列表： 

wx.config({
    jsApiList: ["getLocation"]
})
