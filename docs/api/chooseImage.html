使用 

uni.$uniQuery.chooseImage.then(res=> {
    // 获取成功
})catch(err=>{
    // 获取失败
});

其中，res的数据格式如下： 

// 数组的条目是base64图片地址
Array<string>

权限相关

微信公众号

需要在 jsApiList 补充如下方法列表： 

wx.config({
    jsApiList: ["chooseImage", "getLocalImgData"]
})
