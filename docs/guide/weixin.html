微信小程序

目前微信小程序使用一些比如定位、保存图片到相册等功能的时候，都需要提示用户同意隐私协议。

在首页加入下面的代码即可：

0.2.0 新增 
<template>
        
    
    
    

<template>
<script>
    export default {
        methods: {
            doAgree() {
                console.log("用户同意隐私授权, 接下来可以调用隐私协议中声明的隐私接口");
            },
            doDisagree() {
                console.log("用户拒绝隐私授权, 未同意过的隐私协议中的接口将不能调用");
            }
        },
    };
</script>
