如何安装和引入？

安装

我们使用npm的方式进行发布和管理版本： 

npm install --save uni-query

引入

我们推荐使用 easycom 的方式引入组件： 

"easycom": {
    "autoscan": true,
    "custom": {
        "^ui-(.*)": "uni-query/src/components/$1/index.vue"
    }
}

此外，还需要进行安装： 

import { createApp } from 'vue';
import uniQuery from 'uni-query';

let app = createApp(App);
app.use(uniQuery, options = {});
温馨提示：其中 options 可选，用于配置一些个性化或扩展内容，具体可配置内容，我们将在用到的地方进行说明。

最后，在./src/uni.scss中完成主题引入（2.0.0 新增）： 

@import 'uni-query/src/theme/default.scss';

至此，就可以在项目中正常使用了。
