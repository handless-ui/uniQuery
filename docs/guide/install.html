如何安装和引入？

安装

我们使用npm的方式进行发布和管理版本： 

npm install --save uni-query

引入

我们推荐使用 easycom 的方式引入组件： 

"easycom": {
    "autoscan": true,
    "custom": {
        "^ui-(.*)": "uni-query/components/$1/index.vue"
    }
}

此外，还需要对API根据不同的vue版本安装：

vue2.js 

import Vue from 'vue';
import uniQuery from 'uni-query/libs/index';

Vue.use(uniQuery, options = {});

vue3.js+ 

import { createApp } from 'vue';
import uniQuery from 'uni-query/libs/index';

let app = createApp(App);
app.use(uniQuery, options = {});

至此，就可以在项目中正常使用了。

温馨提示：其中 options 可选，用于配置一些个性化或扩展内容，具体可配置内容，我们将在用到的地方进行说明。
